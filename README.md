# Todo Application

A full-stack Todo application built with Flask, SQLAlchemy, and Bootstrap. This application provides a RESTful API backend and a responsive frontend interface for managing todo items.

## Features

- Create, Read, Update, and Delete todo items
- Mark todos as complete/incomplete
- Responsive design using Bootstrap
- RESTful API backend
- Request logging
- SQLite database
- Interactive frontend with real-time updates

## Tech Stack

### Backend
- Flask
- SQLAlchemy
- SQLite

### Frontend
- Bootstrap 5
- Font Awesome
- JavaScript (Vanilla)

## Prerequisites

- Python 3.8 or higher
- pip (Python package manager)
